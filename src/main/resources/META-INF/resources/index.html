Financeiro API!

This API is made using Quarkus.

What is this?

This is a financial management API service. It offers an integrated and updated view of the financial situation, facilitating management, planning, and decision-making.

OpenAPI & Swagger-UI Documentation

This service leverages the OpenAPI specification alongside Swagger-UI, providing a dynamic, interactive documentation experience. With Swagger-UI, users can:

  • Visualize and interact with the API's resources without any implementation logic in place.
  • Test API endpoints with built-in request builders.
  • User-friendly interface that simplifies both backend implementation and client-side consumption.
